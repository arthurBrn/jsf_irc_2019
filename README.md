### Installation
```sh
$ git clone git@gitlab.com:baronbrn/jsf_irc_2019.git
$ cd jsf_irc_2019
$ ./script_front/install_front.sh
$ ./script_front/launch_server.sh
```
> Now you can go to 127.0.0.1:4200
